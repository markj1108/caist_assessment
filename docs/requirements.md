## Task & Project Tracker - Requirements
# 1. Problem Statement
    It is hard for a team to track their tasks across multiple projects. The Task & Project Tracker aims to provide a simple, centralized system that let you add your projects and break them down into tasks, assign roles, set deadlines, and monitor progress in real time.

# 2. User Personas
- Team Leader 
    - they can create projects, assign roles, set deadlines, and monitor progress in real time.
- Member 
    - see their assigned roles and mark their progress.
- Admin
    - edit their roles, enable/disable their account

# 3. User Stories
- As a Team Leader, I want to create multiple projects and break them down into tasks to distribute to my members.
- As a Team Leader, I want to set priorities on some projects so they know what to finish first.
- As a Team LEader, I want to see status bar on every projects to know the progress of it.
- As a Member, I want to see all task assigned to me so that I can focus on my work.
- As a Member, I want to update task status(To Do, In Progress, Completed) so that the progress is visible.
- As a Member, I want to receive reminders for upcoming or overdue tasks so that I donâ€™t miss deadlines.

# 4. Acceptance Criteria
- Team Leader can add tasks to a project with a title, description, due date, priority, and status.
- Task status updates are reflected immediately in project views.
- Members can view a list of tasks assigned to them.
- The system displays task and project progress
- The interface is responsive and usable on desktop.
